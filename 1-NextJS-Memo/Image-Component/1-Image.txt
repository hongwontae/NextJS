1. NextJS에서 최적화 이미지 관리를 위해 제공되는 컴포넌트이다.


2. 주요 기능
1. 자동 최적화 : 해상도에 맞춰 이미지를 동적으로 리사이징, 캐싱, 이미지 크기 줄이기
2. 사이즈 조정 
3. 레이징 로딩 => default props, 이미지가 화면 상에 필요할 떄 로드되고 렌더링된다.
4. 모든 이미지 포맷 지원
5. Responsive 이미지


3. props
3-1. priority : 이미지를 우선 로드한다. 이 뜻은 이미지 로드를 우선한다는 뜻이지 이미지를 가져오기전에 해당 파일 코드를 막는 건 아니다.
3-2. fill : 이미지가 부모 요소의 크기에 맞춰 채워지도록 한다.
