4. 서버 컴포넌트가 비동기적으로 데이터를 가져온다는 게 무슨 의미일까?
4-1. 서버 컴포넌트는 async 키워드를 컴포넌트 자체가 사용할 수 있다.
4-2. 그렇게 되면 비동기 함수의 실행 결과를 await하여 가져오고 HTML을 구성한다. (서버 사이드 렌더링)
4-3. 이후 해당 HTML을 건네준다.


5. NextJS - SQLite
1. sqlite을 사용해서 원하는 데이터를 가져오는 함수를 만든다.
2. db.prepare('sql') => 원하는 데이터를 가져오는 SQL을 문자열 인자로 넣는다.
3. prepare().all() => SQL 문을 통해 가져온 모든 데이터를 반환한다.
4. 비동기 함수가 아니다. Promise을 반환하지 않는다.


6. Loading Page
6-1. loading이라고 이름이 붙인 컴포넌트는 해당 페이지 HTML이 도착하기 전에 보여주는 HTML이다.
6-2. 주로 서버 컴포넌트 비동기 작업으로 데이터를 가져오고 HTML을 구성하는 시간이 다소 걸릴 떄 유용하다.


7. <Suspense>
7-1. 비동기적으로 로딩되는 컴포넌트를 감싸고 해당 컴포넌트가 렌더링되기 전까지 fallback을 보여주는 컴포넌트
7-2. <Suspense>에 감싸지는 컴포넌트는 내부에 비동기로직이 존재하는 완성된 컴포넌트여야 한다.
