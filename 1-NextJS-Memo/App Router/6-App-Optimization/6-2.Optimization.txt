1. MetaData

1-1. 정적 페이지의 MetaData
=> export const metadata = {} : 정적 metadata는 이 형식을 갖춰야 한다.
=> title : 페이지 제목 (탭 name, 검색 결과 name)
=> description : 페이지 설명 (구글 검색 결과 밑에 설명처럼 되어 있는 것을 의미한다.)

1-2. 동적 페이지의 MetaData
=> export async function generateMetadata(){} : 동적 metadata는 이 형식을 갖춰야 한다.
=> 원하는 결과를 얻기 위해 작업을 하고 객체를 return 한다.
=> 해당 객체의 key는 title, description..등을 사용해서 value를 넣어주면 된다.

** <title> 태그에 title, <meta> 태그에 description이 적용된다.

1-3. 레이아웃 Metadata
=> 레이아웃에 정적 페이지 metadata을 규정하는 것처럼 metadata를 설정할 수 있다.
=> 해당 메타 데이터는 메타 데이터를 사용하지 않는 page.js의 metadata를 설정한다.
=> page.js에 metadata가 설정되어 있다면 layout metadata는 후순위이다.
=> 즉, page.js의 metadata >> layout의 metadata
